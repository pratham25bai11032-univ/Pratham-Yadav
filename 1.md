## Project overview

This repository is a **Digital Literacy Portfolio** for the course **CSE0001 – Digital Literacy**. The scenario: you act as a **Student Digital Ambassador**, helping peers navigate the digital world—professional online profiles, safe and responsible communication, and practical digital skills.

The portfolio documents **five tasks** (one per module), each with artifacts in the folders below and reflections in the project report. The GitHub repository itself is part of the graded deliverable and can remain in your long-term digital presence.

**Final submission (per brief):**

- This README (overview, identity, module summaries, links)
- `report/Project_Report.pdf` or `.docx`
- Folders: `task-1-presentation/`, `task-2-portfolio/`, `task-3-platforms/`, `task-4-email-etiquette/`, `task-5-cybercrime/`

**Submission:** Share the **public GitHub repository URL** via the **VITyarthi** portal.

## Module summaries

| Module | Task | What you deliver (summary) |
|--------|------|----------------------------|
| **Module 1** | Digital literacy awareness infographic | One-page visual (e.g. Canva, Prezi, Miro) covering **at least 3 of:** what digital literacy is, useful tools for students, safe internet practices, professional online presence, email etiquette. Export/screenshot → `task-1-presentation/`. Report: 150–200 words on tool, design, and reflection. |
| **Module 2** | Student digital portfolio | **≥3 platforms** among GitHub, LinkedIn, Kaggle, Stack Overflow, ResearchGate. **GitHub:** profile README repo with name, branch, year, one learning goal. **LinkedIn:** education filled. Screenshots → `task-2-portfolio/`. Report: 150–200 words on choices and 4-year plan. |
| **Module 3** | Coding & collaboration platforms | **Part A:** Account on HackerRank, CodeChef, or HackerEarth + complete ≥1 beginner challenge; screenshot → `task-3-platforms/`. **Part B:** Google Form with **exactly 5** quiz questions (incl. multiple choice + short answer); share form link **in this README**; screenshots of form + linked Sheet → `task-3-platforms/`. Report: 150–200 words. |
| **Module 4** | Professional email & etiquette | **Two emails:** (1) to professor—extension request; (2) to internship coordinator—interest. Clear subject, greeting, body, sign-off → `task-4-email-etiquette/`. **Checklist:** ≥5 Do’s and ≥5 Don’ts for social media → `social-media-checklist.md` or `.txt`. Report: 100–150 words on a poor-communication scenario. |
| **Module 5** | Cybercrime awareness | **Case study** (200–250 words): phishing, UPI/payment fraud, identity theft, cyberbullying, or ransomware—how it happens, targets, consequences → `casestudy.md` or `.pdf`. **Prevention:** ≥8 actionable tips (≥2 for **financial/UPI**), incl. reporting: **National Cyber Crime Portal** and **1930** → `prevention-checklist.md` or `.txt`. Report: 100–150 words reflection. |

**Assessment:** 20 marks per task; **+5 bonus** for repository structure, files, and README completeness. **Pass:** 50/100.
